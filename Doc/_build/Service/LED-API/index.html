テープLED API ‘Elpis-LED-API’ [2014/07/22]¶

§0 本資料について¶

　本資料は、ArtifactNoiseが制作した資料であり、一部開発中の物も含まれている場合があります。

　最新の情報、ご質問等は info@ArtifactNoise.com までご連絡ください。

§1.概要¶

　API側からリアルのテープLEDの色を設定できます。

　設定した色はその色通りに光、部屋一面の演出が可能です。

　また、テープ状ですので、アクリルに照らせば看板も作成出来ます。

パラメータ 値 説明 バージョン 備考

(4) 戻り値¶

戻り値に関しては、JSON形式になります。

パラメータ 値 説明 バージョン 備考

(5) サンプルコード¶

POST

Return

{“state”,”OK”}
